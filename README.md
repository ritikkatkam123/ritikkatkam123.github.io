# Portfolio Project

## Project info

Personal portfolio website built with React + Vite.

## Run locally

```sh
# Install dependencies
npm install

# Start local dev server
npm run dev
```

Default local URL: `http://localhost:8080`

## Build

```sh
# Production build
npm run build

# Preview production build
npm run preview
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
